第一个程序

  • 代码
  1. #include <iostream>，表示包含一个名为iostream的头文件。该头文件表示C++的标准头文件，io表示输入输出，stream表示流。即：输入输出流的头文件
  2. using namespace std;指定一个空间，该空间的名字为std（standard），C++标准库中的所有组件都是在这个空间中被声明和定义的，相当于作用域在该空间处。在编写C++程序的时候必须先声明空间名std
  3. main函数不多说，学过C了
  4. cout<<"你好，C++！\n";cout是一个输出流对象，是C++系统预定义的的对象。“<<”是输出操作符，之后紧跟双引号和要输出在屏幕中的字符，
  5. system("pause")表示暂停。使用后程序运行到该语句的时候就会出现请按任意键继续···如果没有改行代码，在某些c++编译器程序运行时屏幕会一闪而过
  6. return不多说

sizeof函数

  • sizeof函数可以用来获取各个数据类型的字节长度，例如获取long型字节长度cout<<sizeof(long);
  • cout<<"bool的字节长度为"<<sizeof(bool)<<endl,这里有多个**<<**,表示连续输出，如果末尾不再输出，应该加endl，这样执行下一个输出的时候就会换行。不加endl就是不换行。

输出

  • 两个整数相加，输入两个整数，计算两个整数相加的和
  1. 第10行的 cin>>a>>b;中cin表示键盘（标准输入设备），该行代码表示从键盘中先后输入两个值，把先输入的值赋给a，后输入的值赋给b。
  2. 从键盘输入数据时，需要严格按照要求。1 2，1和2之间可以有多个空格或1个空格，但是不能有其他符号。如：1,2
  3. 如果输入数字时，错误的输入了其他符号
  1. 输入、输出应该严格按照格式来写

  • c++程序对用到的所有数据都必须指定其数据类型。
  • 程序中要预先设置初值，C++允许定义变量的同时初始化变量。
