MISC

签到！DK 盾！

  • 直接关注微信公众号，发送BaseCTF2024,得到flag:BaseCTF{2024_sp0n5ored_by_dkdun}

基础

  • 给了一个题目附件

  • 观察看到后面是等于号填充，先想到Base加密，再看到有字母和数字，去在线base解密网站Base16 Base32 Base58 Base62 Base64 Base85 Base91 编码/解码 - 锤子在线工具 (toolhelper.cn)
  • 根据密文的字符和编码表相对应，得到很可能是base32编码，解码之后又得到QmFzZUNURnt3ZTFjMG1lX3RvX2I0c2VjdGZ9

  • 根据观察发现是base64编码，然后使用base64解码得到flag，BaseCTF{we1c0me_to_b4sectf}

人生苦短我用py

  • 这题也就是代码审计，加上一些明文爆破即可得到flag,得到flagBaseCTF{s1Mpl3_1s_BeTt3r_Th4n_C0mPl3x}

  • 在abort(15)中先要读懂代码，然后再进行逆过程运算(读懂代码可以借助AI)

  • 在abort(18)这个if语句中使用明文爆破

根本进不去啊

  • 根据提示要用到域名解析

    • dig（Domain Information Groper）是一个用来查询 DNS 记录的命令行工具，可以用来查询域名系统（DNS）中的各种记录，包括 A 记录（将域名解析到 IP 地址）、TXT 记录（可以用于存储任意文本数据）、MX 记录（邮件交换记录）等。

  • 直接在Linux终端上输入命令dig flag.basectf.fun TXT,即可得到flag。BaseCTF{h0h0_th1s_15_dns_rec0rd}

海上遇到鲨鱼

  • 简单的wireshark流量抓包分析题目

  • 使用wireshark打开附件，会出现如下图所示的流量

  • 此题流量比较少，从头开始看也能看得到。这里就直接定位HTTP协议位置，即可看到flag，当时网络传输都是大端序，而不是小端序，所以，应该把flag翻转一下

  • 使用Python脚本翻转字符串,得到flagBaseCTF{15ef386b-a3a7-7344-3b05-ac367316fb76}

复现：正着看还是反着看呢？

  • 首先先了解一下jpg（jpeg）的二进制文件头

    • JPEG图片的二进制头标识由两个字节组成，分别是0xFF和0xD8，这标志着JPEG文件的开始。

  • 题目给了个flag的二进制文件，但是没有文件后缀名，并不知道文件类型

  • 所以先将flag拖入010editor,发现头两个字节是0x00 0x00，尾两个字节是 0xD8 0XFF

  • 而0xD8 0xFF恰好是jpg二进制文件头的逆序，这样就可以想到该二进制文件是将jpg图片文件二进制倒序输出
  • 这时编写脚本将该图片复原
  • 然后再将还原后的图片使用010editor打开，然后翻到文件二进制的末尾，发现隐藏了一个flag.txt文件
  • 然后使用工具提取出来flag
    • 这里要使用两个工具binwalk和foremost

  • 输入两个命令后就可以得到一个文件夹，这里面有个压缩包，压缩包里面放的就是flag

  • 得到flag BaseCTF{h3ll0_h4cker}

复现：你也喜欢圣物吗

  • 题目给了两个附件分别是png图片和压缩包，压缩包需要密码才能打开
  • 压缩包里面有

WEB

暗室

  • 这题算是WEB题的签到题目了。直接F12即可得到flagBaseCTF{1510e990-b8b1-490f-a5db-6886b7f69950}

CRYPTO

会算md5吗

  • 查看题目附件,得到的是flag中每个字符直接使用md5加密
  • 直接去md5在线解密网站解密即可md5在线解密破解,md5解密加密 (cmd5.com),相同的字符md5加密后值也是相同的，就不要复制粘贴那么多次了。最后得到flagBaseCTF{a4bf43a5-3ff9-4764-bda2-af8ee4db9a8a}

ez_math

babypack

babyrsa

  • 简单的rsa加密，直接就模版题目
  • exp:得到flag b'BaseCTF{7d7c90ae-1127-4170-9e0d-d796efcd305b}'

十七倍

  • 题目附件重点如下，由于是字节类型，所以最高是255，也就是说乘17后会出现溢出，这种溢出的情况相当于乘17模256，所以了解乘法逆元后即可得到

  • 然后之前乘法逆元求解flag，得到flag BaseCTF{yoUr_CrYpt0_1earNinG_5tarTs_n0w}

helloCrypto

  • 签到题，没难度，运行py文件就可以得到flag BaseCTF{b80bf679-1869-4fde-b3f9-d51b872d31fb}

ez_rsa

  • rsa加密的简单变式
  • 解方程得到p和q,已知p+q和p*q直接解方程，使用sage解方程
  • 然后再解密得到flag b'BaseCTF{it_1s_ez!!}'

REVERSE

You are good at IDA

  • 下载IDA，并学会使用IDA
  • 该题完全是找字符串,找到flag的一部分Y0u_4Re_,并且知道提示我们按shift f12

  • 按后进入字符串界面，然后双击

  • 得到第二部分900d_47_

  • 然后最后一个部分

  • 最后得到完整的flag Base{Y0u_4Re_900d_47_id4}

UPX mini

  • 直接在上网下载upx脱壳工具，然后upx-d即可，脱壳后使用Ida打开,发现又是一个base编码。直接在线解码

  • 得到flag BaseCTF{Hav3_@_g0od_t1m3!!!}

PWN

签个到吧

  • 直接nc即可

echo

  • nc连接进去，使用这个命令echo *可以看到目录文件

  • 然后使用while read line; do echo $line; done < /flag去读取flagBaseCTF{4caa87fa-30fc-4c82-845b-03d3b53d2853}

  • 当然官方也有一种读取flag的命令

Ret2text

  • 入门题目不多讲

shellcode_level0

  • ret2shellcode，输入一定长度的shellcode然后执行，直接得到shell。
  • 不多赘述
